# rstTools README

Сборник инструментов для работы с разметкой reStructuredText.

## Возможности

Какие инструменты доступны:

1. Мастер управления переменными:
   
   - Подставляет инлайновые инклюды из файлов, которые подключены в файле `conf.py` проекта или определены в самом файле. 
     Начните вводить значение переменной для отображения списка доступных переменных.

   - Подсвечивает значение переменной при наведении на нее.

   - Позволяет перейти к редактированию переменной. Для этого нужно кликнуть на переменную с зажатой клавишей `Ctrl`.
 
2. Мастер управления ссылками внутри документа:

   - Позволяет подставлять пути к файлам внутри роли `doc`.
     Начните вводить путь для отображения доступных файлов.

   - Подсвечивает некорректные ссылки, в которых путь к файлу указан неправильно.
