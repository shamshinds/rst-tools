# rstTools README

Сборник инструментов для работы с разметкой reStructuredText.

## Возможности

Какие инструменты доступны:

1. Управления переменными:
   
   - Подставляет инлайновые инклюды (переменные) из файлов, которые подключены в файле `conf.py` проекта или определены в самом файле. 
     Начните вводить `|` и значение переменной для отображения списка доступных переменных.

   - Показывает значение переменной и ее расположение при наведении на нее.

   - Позволяет открыть картинку, котора.

   - Подсвечивает переменные, которые не удалось определить.

   - Позволяет перейти к редактированию переменной.
     Для этого нужно кликнуть на переменную с зажатой клавишей `Ctrl`.
 
2. Управление ссылками внутри документа:

   - Позволяет подставлять относительные пути к файлам проекта внутри роли `doc`.
     Начните вводить путь для отображения доступных файлов.

   - Позволяет подставлять пути к файлам другого проекта внутри роли `doc`.
     Начните вводить название проекта для отображения доступных проектов.
   
   - Показывает путь к файлу в ссылке, заголовок этого файла и его расположение при наведении на ссылку.

   - Подсвечивает некорректные ссылки, в которых путь к файлу указан неправильно.

3. Управление инклюдами:
   
   - Показывает расположение файла с инклюдом при наведении на него.

   - Позволяет перейти к редактированию фрагмента из инклюда.
     Для этого нужно кликнуть на путь к файлу с зажатой клавишей `Ctrl`.
